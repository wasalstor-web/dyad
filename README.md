# VPS Connection Template

This repository contains configuration files for setting up remote development on a VPS.

## What's Included

- `.devcontainer/devcontainer.json` - Development container configuration
- `.vscode/settings.json` - VS Code workspace settings  
- `VPS_CONNECTION.md` - Comprehensive guide for VPS setup and connection

## Quick Start

See [VPS_CONNECTION.md](./VPS_CONNECTION.md) for detailed setup instructions.

## Remote Development Options

1. **VS Code Remote - SSH**: Connect directly to your VPS
2. **Dev Containers**: Use containerized development environment
3. **Direct SSH**: Traditional SSH-based development

For full documentation, see the [VPS Connection Guide](./VPS_CONNECTION.md).
